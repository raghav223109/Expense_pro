
# Expense Tracker Pro

Expense Tracker Pro is a modern Flask-based personal finance management application with advanced analytics, authentication, admin dashboard, savings goals, export reports, and multi-currency support.

## Features

- User Authentication
- Expense Tracking
- Savings Goals
- Doughnut Analytics Charts
- Admin Dashboard
- Multi Currency Support
- CSV Export
- Dark Dashboard UI
- Responsive Design

## Technologies Used

- Python
- Flask
- SQLite
- SQLAlchemy
- Tailwind CSS
- Chart.js
- Pandas

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
